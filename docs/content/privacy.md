The `kdlctl` documentation site is a static Hugo site. By default it does not collect personal data, run a custom analytics stack, or require account sign-in to read the documentation.

When you follow outbound links from this site, the destination service may apply its own privacy policy. That includes GitHub, Coveralls, Google Cloud documentation, and any other third-party sites referenced in the docs.

If this site is later configured with analytics, forms, or feedback tooling, the documentation and repository policy files should be updated to describe that data handling explicitly.
